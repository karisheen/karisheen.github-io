Coronavirus Stat Tracker Voice Assistant

  • Source code
  • Created a voice assistant using python that allows users to ask coronavirus related questions (i.e., number of cases and number of deaths by country).
  • Scraped covid-19 data from a website using ParseHub software.
  • Used the pyttsx3 library to convert text to speech.
  • Used the speech_recognition library to convert speech to text.
